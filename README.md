# arbitrager

connecting up to 1024 orderbooks via websocket and scans for arbitrage opportunities
on varoius popular exchanges.


**STRATEGIES SUPPORTED**

Triangular arbitrage - *checking all possible trade routes*

**EXCHANGES SUPPORTED**

Binance
